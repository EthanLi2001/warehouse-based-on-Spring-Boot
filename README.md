# 🏭 智能仓储管理系统（Spring Boot）

## 📌 项目简介
本项目是一个基于 Spring Boot 的智能仓储管理系统，实现了商品入库、出库、库存管理等核心功能，适用于中小型仓储管理场景。

---

## 🧠 技术栈
- Spring Boot
- MyBatis / MyBatis-Plus
- MySQL
- HTML + Thymeleaf
- Maven

---

## ⚙️ 功能模块

### 👤 用户模块
- 登录 / 登出
- 用户权限管理

### 📦 商品管理
- 商品新增 / 修改 / 删除
- 商品分类管理

### 📊 仓库管理
- 入库管理
- 出库管理
- 库存查询
